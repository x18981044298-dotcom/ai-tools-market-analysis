# 🤖 AI工具市场深度分析报告

基于 **There's An AI For That** 数据的交互式市场分析仪表板。

## 📊 分析维度

- **🏠 执行摘要** - 市场全景与核心KPI
- **🚀 市场结构视角** - 供给密度、层级结构、粒度差异
- **🧭 用户需求视角** - 用户场景、意图类型、竞争强度
- **🔎 趋势机会视角** - LLM驱动、痛点驱动、机会象限
- **🪜 产品机会视角** - 高/低竞争赛道、纵深/横切机会
- **🧱 分类系统视角** - 分类合理性、粒度一致性、超级领域
- **🧲 商业化视角** - 目标用户、商业模式、变现策略
- **🧬 用户角色视角** - 工作/兴趣/情感人设分析

## 🚀 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📈 数据来源

数据基于 [There's An AI For That](https://theresanaiforthat.com/) 网站的AI工具分类统计。
